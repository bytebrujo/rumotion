# @picus/cli

Control Picus features using the `npx picus` command

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/cli.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/cli?minimal=true)

## Installation

```bash
npm install @picus/cli --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/cli) for more information.
