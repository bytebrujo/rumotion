# @picus/openai-whisper

Work with the output of the OpenAI Whisper API

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/openai-whisper.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/openai-whisper?minimal=true)

## Installation

```bash
npm install @picus/openai-whisper --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/openai-whisper) for more information.
