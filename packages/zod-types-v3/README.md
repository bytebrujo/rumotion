# @picus/zod-types-v3

Zod 3.22.3 types for Picus

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/zod-types-v3.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/zod-types-v3?minimal=true)

## Installation

```bash
npm install @picus/zod-types-v3 --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.
