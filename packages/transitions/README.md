# @picus/transitions

Library for creating transitions in Picus

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/transitions.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/transitions?minimal=true)

## Installation

```bash
npm install @picus/transitions --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/transitions) for more information.
