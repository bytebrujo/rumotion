# @picus/eslint-config

Default configuration for Picus templates (ESLint <= 8)

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/eslint-config.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/eslint-config?minimal=true)

## Installation

```bash
npm install @picus/eslint-config --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/brownfield#install-the-eslint-plugin) for more information.
