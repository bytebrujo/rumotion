# @picus/captions

Primitives for dealing with captions

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/captions.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/captions?minimal=true)

## Installation

```bash
npm install @picus/captions --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://picus.dev/docs/captions/api) for more information.
