# @picus/cloudrun

Render Picus videos on Google Cloud Run

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/cloudrun.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/cloudrun?minimal=true)

## Installation

```bash
npm install @picus/cloudrun --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/cloudrun) for more information.
