# @picus/install-whisper-cpp

Helpers for installing and using Whisper.cpp

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/install-whisper-cpp.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/install-whisper-cpp?minimal=true)

## Installation

```bash
npm install @picus/install-whisper-cpp --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/install-whisper-cpp) for more information.
