# @picus/motion-blur

Motion blur effect for Picus

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/motion-blur.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/motion-blur?minimal=true)

## Installation

```bash
npm install @picus/motion-blur --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/motion-blur) for more information.
