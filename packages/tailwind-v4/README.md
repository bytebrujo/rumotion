# @picus/tailwind-v4

Enable TailwindCSS support in Picus (TailwindCSS v4)

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/tailwind-v4.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/tailwind-v4?minimal=true)

## Installation

```bash
npm install @picus/tailwind-v4 --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/tailwind/tailwind) for more information.
