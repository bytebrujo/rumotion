# @picus/webcodecs

Media conversion in the browser

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/webcodecs.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/webcodecs?minimal=true)

## Installation

```bash
npm install @picus/webcodecs --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.
