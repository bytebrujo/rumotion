# @picus/animation-utils

Helpers for animating CSS properties

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/animation-utils.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/animation-utils?minimal=true)

## Installation

```bash
npm install @picus/animation-utils --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/animation-utils/) for more information.
