# @picus/lottie

Include Lottie animations in Picus

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/lottie.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/lottie?minimal=true)

## Installation

```bash
npm install @picus/lottie --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

See the [documentation](https://www.picus.dev/docs/lottie) for more information.
