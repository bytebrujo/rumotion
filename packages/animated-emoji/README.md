# @picus/animated-emoji

Google Fonts Animated Emojis as Picus components

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/animated-emoji.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/animated-emoji?minimal=true)

## Installation

```bash
npm install @picus/animated-emoji --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.
