# @picus/google-fonts

Use Google Fonts in Picus

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/google-fonts.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/google-fonts?minimal=true)

## Installation

```bash
npm install @picus/google-fonts --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/google-fonts) for more information.
