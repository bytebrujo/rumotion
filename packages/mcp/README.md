# @picus/mcp

Picus's Model Context Protocol

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/mcp.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/mcp?minimal=true)

## Installation

```bash
npm install @picus/mcp --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.
