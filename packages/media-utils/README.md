# @picus/media-utils

Utilities for working with media files

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/media-utils.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/media-utils?minimal=true)

## Installation

```bash
npm install @picus/media-utils --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/media-utils) for more information.
