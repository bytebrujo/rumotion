# @picus/lambda

Render Picus videos on AWS Lambda

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/lambda.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/lambda?minimal=true)

## Installation

```bash
npm install @picus/lambda --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/lambda) for more information.
