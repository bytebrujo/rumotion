# @picus/babel-loader

Babel loader for Picus

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/babel-loader.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/babel-loader?minimal=true)

## Installation

```bash
npm install @picus/babel-loader --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/legacy-babel) for more information.
