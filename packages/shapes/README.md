# @picus/shapes

Generate SVG shapes

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/shapes.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/shapes?minimal=true)

## Installation

```bash
npm install @picus/shapes --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.
