# @picus/gif

Embed GIFs in a Picus video

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/gif.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/gif?minimal=true)

## Installation

```bash
npm install @picus/gif --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/gif) for more information.
