# @picus/player

React component for embedding a Picus preview into your app

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/player.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/player?minimal=true)

## Installation

```bash
npm install @picus/player --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/player) for more information.
