# @picus/layout-utils

Utilities for working with layouts

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/layout-utils.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/layout-utils?minimal=true)

## Installation

```bash
npm install @picus/layout-utils --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/layout-utils) for more information.
