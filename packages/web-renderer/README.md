# @picus/web-renderer

Render videos in the browser (not yet released)

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/web-renderer.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/web-renderer?minimal=true)

## Installation

```bash
npm install @picus/web-renderer --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

See the [documentation](https://www.picus.dev/docs/web-renderer/) for more information.
