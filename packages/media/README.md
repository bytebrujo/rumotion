# @picus/media

Experimental WebCodecs-based media tags

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/media.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/media?minimal=true)

## Installation

```bash
npm install @picus/media --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://picus.dev/docs/media) for more information.
