# @picus/studio

APIs for interacting with the Picus Studio

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/studio.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/studio?minimal=true)

## Installation

```bash
npm install @picus/studio --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/studio/api) for more information.
