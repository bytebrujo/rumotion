# @picus/zod-types

Zod types for Picus

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/zod-types.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/zod-types?minimal=true)

## Installation

```bash
npm install @picus/zod-types --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

See the [documentation](https://www.picus.dev/docs/zod-types) for more information.
