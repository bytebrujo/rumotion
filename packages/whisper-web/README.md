# @picus/whisper-web

Helpers for using Whisper.cpp in browser using WASM

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/whisper-web.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/whisper-web?minimal=true)

## Installation

```bash
npm install @picus/whisper-web --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

See the [documentation](https://www.picus.dev/docs/whisper-web) for more information.
