# @picus/vercel

Render Picus videos on Vercel Sandbox

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/vercel.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/vercel?minimal=true)

## Installation

```bash
npm install @picus/vercel --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/vercel/api) for more information.
