# @picus/preload

Preloads assets for use in Picus

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/preload.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/preload?minimal=true)

## Installation

```bash
npm install @picus/preload --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/preload) for more information.
