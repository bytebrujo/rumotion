# @picus/media-parser

A pure JavaScript library for parsing video files

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/media-parser.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/media-parser?minimal=true)

## Installation

```bash
npm install @picus/media-parser --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/media-parser) for more information.
