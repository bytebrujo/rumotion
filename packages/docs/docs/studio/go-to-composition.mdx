# goToComposition()<AvailableFrom v="4.0.287"/>

Selects a composition in the Picus Studio.

## Examples

```tsx twoslash title="Go to a composition"
import {goToComposition} from '@picus/studio';

goToComposition('my-composition');
```

If the composition is not found, an error will be thrown.
