# pause()<AvailableFrom v="4.0.287"/>

Pauses playback in the Picus Studio.

## Examples

```tsx twoslash title="Pause on button click"
import {pause} from '@picus/studio';

const PauseButton = () => {
  return <button onClick={pause}>Pause</button>;
};
```

```tsx twoslash title="Pause programmatically"
import {pause} from '@picus/studio';

// Pause playback
pause();
```
