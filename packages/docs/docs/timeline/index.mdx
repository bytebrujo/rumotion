# &lt;Timeline&gt;

<img alt="Picus Timeline Banner" src="/img/picus-timeline-banner.jpg" />
<br />
<br />

A paid customizable timeline component that integrates seamlessly with [Picus Player](/docs/player).  
It's perfect for builders who need a layer-based editing interface for their video editing app.

You can buy the timeline component on [picus.pro](https://www.picus.pro/timeline).

<TableOfContents />

## Key Features

Interface:

- Multiple track support with drag-and-drop functionality (between and within tracks)
- Timeline item selection and manipulation
- Keyboard shortcuts for common actions
- `CanvasComposition` for rendering the timeline state in the Picus Player
- Zoomable timeline and `<ZoomSlider>` component to control the zoom level

Customization:

- Theming with variables (Tailwind CSS V4 and V3)
- Extensible state management
