_available since v4.0_

In order to use `ffprobe` without having to directly install it, Picus provides it via `npx picus ffprobe`.

Note that in order to keep the binary size small, the FFprobe binary only understand the codecs that Picus itself supports: H.264, H.265, VP8, VP9 and ProRes. A binary from the 7.1 release line of FFprobe is used.

# Example

```bash
npx picus ffprobe your_video.mp4
```

To find out more about FFprobe, visit their [docs](https://ffmpeg.org/ffprobe.html).
