Prints out which versions of Picus packages are installed.

Also checks if `zod` and [`mediabunny`](/docs/mediabunny/version) are installed with the correct versions.

```
npx picus versions
```

In the optimal case, your versions are aligned and the output of the command looks like this:

```
On version: 3.0.19
- @picus/bundler
- @picus/cli
- @picus/eslint-config
- @picus/renderer
- @picus/media-utils
- @picus/babel-loader
- @picus/lambda
- @picus/three
- @picus/gif
- picus

✅ Great! All packages have the same version.
```
