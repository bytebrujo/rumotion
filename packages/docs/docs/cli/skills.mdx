Install or update [Agent Skills](/docs/ai/skills) from `picus-dev/skills`.

```
npx picus skills <subcommand>
```

## Subcommands

### `add`

Install skills from `picus-dev/skills`:

```bash
npx picus skills add
```

### `update`

Update skills from `picus-dev/skills`:

```bash
npx picus skills update
```

## What are skills?

Agent Skills are instructions that define best practices for working in Picus projects. These skills are useful for AI agents like Claude Code, Codex, or Cursor.

Skills are installed to the `.claude/skills` directory in your project.
