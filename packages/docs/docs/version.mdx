You may import this constant to get the current version of Picus.

Only the version of the [`picus`](/docs/picus) package will be reported.  
A [version conflict](/docs/cli/versions) with other Picus packages cannot be ruled out.

```ts twoslash title="Importing VERSION from picus"
import { VERSION } from "picus";

console.log(VERSION); // "4.0.57";
```

You can also import it from `picus/version` to avoid importing Picus and its dependencies (i.e, `react` and `react-dom`):

```ts twoslash title="Importing VERSION from picus/version"
import { VERSION } from "picus/version";

console.log(VERSION); // "4.0.57";
```
