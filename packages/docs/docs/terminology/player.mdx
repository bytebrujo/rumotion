The Picus Player is the React component [`<Player>`](/docs/player) that can be embedded into a React app.  
It can be used by installing the [`@picus/player`](https://npmjs.org/packages/@picus/player) package into a React app (for example: Create React App, Next.JS, Remix, React Router, Vite).  
It is not the same as the [Picus Studio](/docs/terminology/studio).
