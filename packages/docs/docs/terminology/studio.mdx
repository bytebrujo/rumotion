The [Picus Studio](/docs/studio) is the editor that opens that when you run [`npx picus studio`](/docs/cli/studio).  
It allows for fast editing and playback of [compositions](/docs/terminology/composition).  
It is included in the [`@picus/cli`](https://npmjs.org/packages/@picus/cli) package.

Not to be confused with the [Picus Player](/docs/player).

You can [deploy the Picus Studio](/docs/studio/deploy-server) to render on a VPS.  
Since v4.0.93, the Studio is also embedded in every [Picus Bundle](/docs/terminology/bundle), albeit it does not support rendering.

## Picus Preview

The former name of the Picus Studio, renamed in v4.0.0.  
Since the Preview gained more capabilities than just previewing, the name was not fitting anymore.
