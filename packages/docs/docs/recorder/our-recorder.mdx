The source code of our videos is available in [`picus-dev/our-recorder`](https://github.com/picus-dev/our-recorder).  
This is the instance of the Recorder that is being used to create [Jonnys](https://twitter.com/JNYBGR) and [Picus](https://twitter.com/picus)'s videos.

This repository may have more and newer features, but it is not guaranteed to be stable.
Use it as a reference if you wonder how we made a specific video!
