## Chat server

Talk to the Picus team and other users in the `#recorder` channel in [Discord](https://picus.dev/discord) to discuss with others using the Recorder.

## Bugs

For bugs with the Recorder, file an issue in the [Recorder repo](https://github.com/picus-dev/recorder/issues).
For issues with Picus itself, see here how to [get help](/docs/get-help).
