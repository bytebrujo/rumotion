## `FrameRange`

```tsx twoslash
import type {FrameRange} from '@picus/renderer';
//                ^?
```

- A single number renders only that frame
- A tuple `[start, end]` renders frames from `start` to `end` (inclusive)
- A tuple `[start, null]` renders frames from `start` to the end of the composition<AvailableFrom v="4.0.421" inline />
