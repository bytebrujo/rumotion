Of course we are bummed if you decided not to use Picus Lambda anymore. We are very receptive to feedback if you'd like to share it with us.

If you would like to remove **all Picus Lambda related objects** from your infrastructure, you can follow these steps.

:::warning
This will remove all videos already rendered, and break your programs that are using Picus Lambda to render videos.
:::

## Delete Lambda Functions

You can delete all functions using the following command. The `yes` flag is already included, if you run this, it will delete all functions without confirmation.

```
npx picus lambda functions rmall -y
```

## Delete projects

```
npx picus lambda sites rmall -y
```

## Delete renders and artifacts

Delete all S3 buckets starting with `picuslambda-` from your AWS account.

---

Make sure you have [applied the instructions to all regions](/docs/lambda/region-selection). You have now cleared up your AWS infrastructure from all Picus Lambda-related resources.
