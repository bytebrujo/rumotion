# @picus/zod-types-v3<AvailableFrom v="4.0.426" />

[`@picus/zod-types`](/docs/zod-types) creates schemas using Zod v4 since Picus v4.0.426.

This package is for if you are using Zod v3.22.3 and want to compose functions like [`zColor()`](/docs/zod-types/z-color) with Zod v3.22.3 schemas.

## Installation

<Installation pkg="@picus/zod-types-v3" />

## Usage

```tsx twoslash title="Using @picus/zod-types-v3 with Zod v3"
import {z} from 'zod'; // zod = 3.22.3
import {zColor} from '@picus/zod-types-v3';

export const mySchema = z.object({
  color: zColor(),
});
```

The package exports the same [`zColor()`](/docs/zod-types/z-color), [`zTextarea()`](/docs/zod-types/z-textarea), and [`zMatrix()`](/docs/zod-types/z-matrix) functions as [`@picus/zod-types`](/docs/zod-types), but they return Zod v3.22.3 schema types.

## When to use

Use [`@picus/zod-types`](/docs/zod-types) with Zod v4:

```tsx twoslash
import {z} from 'zod'; // zod = 4.x.x
import {zColor} from '@picus/zod-types';

export const mySchema = z.object({
  color: zColor(),
});
```

Use [`@picus/zod-types`](/docs/zod-types) if you are not nesting schemas:

```tsx twoslash
import {zColor} from '@picus/zod-types';
import {visualControl} from '@picus/studio';

export const myVisualControl = visualControl('my-color', '#fff', zColor());
```

Use `@picus/zod-types-v3` with Zod v3.22.3:

```tsx twoslash
import {z} from 'zod'; // zod = 3.22.3
import {zColor} from '@picus/zod-types-v3';

export const mySchema = z.object({
  color: zColor(),
});
```

Other Zod 3.x.x versions are not officially supported.
