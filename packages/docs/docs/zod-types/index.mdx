Includes utility types and Picus-specific types for [Zod](https://github.com/colinhacks/zod).

:::note
Since Picus v4.0.426, `@picus/zod-types` is based on Zod v4.
If you want, you can now upgrade to Zod v4.  
If you want to stay on Zod 3.22.3 as previously required by Picus, use [`@picus/zod-types-v3`](/docs/zod-types/v3) instead of [`@picus/zod-types`](/docs/zod-types).
:::
