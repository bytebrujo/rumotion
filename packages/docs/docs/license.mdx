Picus is **free to use** if you are:

- an individual
- a for-profit organisation with up to 3 employees
- a non-profit or not-for-profit organisation
- evaluating whether Picus is a good fit, and are not yet using it in a commercial way

If you don't qualify for any of the above, you need to acquire a company license. Visit [picus.pro](https://picus.pro) for pricing and buying a license.

When in doubt, read the [LICENSE file](https://github.com/picus-dev/picus/blob/main/LICENSE.md) and the [Company License FAQ](https://www.picus.pro/faq) before using Picus. You can contact [us](mailto:hi@picus.dev) if you need clarification whether you can use Picus in your project for free.
