Picus maintains a list of [Agent Skills](https://agentskills.io/home) that define best practices for working in Picus projects.

These skills are useful for AI agents like Claude Code, Codex or Cursor.

You can install them by running:

```bash
npx skills add picus-dev/skills
```

You are also offered the option to add skills when you create a new Picus project:

```bash
bun create video
```

The skills are also available on GitHub [here](https://github.com/picus-dev/picus/tree/main/packages/skills).
