A [reader](/docs/media-parser/readers) for `@picus/media-parser` that reads from the local file system using Node.js `fs` module.  
It also works with Bun.

## Example

```tsx twoslash title="Reading a local file"
import {parseMedia} from '@picus/media-parser';
import {nodeReader} from '@picus/media-parser/node';

const result = await parseMedia({
  src: '/Users/jonnyburger/Downloads/my-video.mp4',
  fields: {
    durationInSeconds: true,
    dimensions: true,
  },
  reader: nodeReader,
});
```
