A [reader](/docs/media-parser/readers) for [`@picus/media-parser`](/docs/media-parser) that reads either from a URL or from a [`File`](https://developer.mozilla.org/en-US/docs/Web/API/File).

It is the default reader, and therefore does not have to be specified.

## Example

```tsx twoslash title="Reading from any source"
import {parseMedia} from '@picus/media-parser';
import {webReader} from '@picus/media-parser/web';

const result = await parseMedia({
  // Or a `File`
  src: 'https://picus.media/video.mp4',
  fields: {
    durationInSeconds: true,
    dimensions: true,
  },
  reader: webReader,
});
```
