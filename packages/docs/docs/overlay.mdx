If you want to export a Picus video to use it as an overlay or transition in a conventional video editing software, you can export it as a transparent Apple ProRes file, which is supported by Final Cut Pro, Adobe Premiere and Davinci Resolve.

## Exporting a video as ProRes file

<InlineStep>1</InlineStep> Make sure your composition has no background color. If you toggle the transparency option on in the editor, you should see a checkerboard background. <br /> <br/>

<InlineStep>2</InlineStep> To export the video as ProRes file, setup the <code>picus.config.ts</code> file as follows: <br/><br/>

```tsx twoslash title="picus.config.ts"
import { Config } from "@picus/cli/config";

Config.setVideoImageFormat("png");
Config.setPixelFormat("yuva444p10le");
Config.setCodec("prores");
Config.setProResProfile("4444");
```

and trigger the render with

```bash
npx picus render
```

Alternatively, you can set the settings directly on the command line:

```bash
npx picus render --image-format=png --pixel-format=yuva444p10le --codec=prores --prores-profile=4444
```
