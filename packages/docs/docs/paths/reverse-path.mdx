_Part of the [`@picus/paths`](/docs/paths) package._

Reverses a path so the end and start are switched.

```tsx twoslash
import { reversePath } from "@picus/paths";

const reversedPath = reversePath("M 0 0 L 100 0");
console.log(reversedPath); // "L 100 0 M 0 0"
```

The function will throw if the path is invalid:

```tsx twoslash
import { reversePath } from "@picus/paths";
// ---cut---
reversePath("picus"); // Error: Malformed path data: ...
```
