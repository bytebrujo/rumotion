The `@picus/renderer` package provides APIs for rendering video server-side.
The package is also internally used by the Picus CLI and [Picus Lambda](/docs/lambda).

:::warning
The configuration file has no effect when using these APIs.
:::

## What's the difference between `renderMedia()` and `renderFrames()`?

In Picus 3.0, we added the [`renderMedia()`](/docs/renderer/render-media) API which combines `renderFrames()` and `stitchFramesToVideo()` into one simplified step and performs the render faster. Prefer `renderMedia()` if you can.
