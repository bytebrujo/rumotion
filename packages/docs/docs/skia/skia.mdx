This package provides utilities useful for integrating [React Native Skia](https://github.com/Shopify/react-native-skia) with Picus.

## Installation

Install, `@picus/skia` as well as `@shopify/react-native-skia`.

<Installation pkg="@picus/skia @shopify/react-native-skia" />

:::note
Since Picus `v3.3.93` and React Native Skia `0.1.191`, `react-native-web` is not a dependency anymore. You may remove it from your project.
:::

Also update **all the other Picus packages** to have the same version: `picus`, `@picus/cli` and others.

:::note
Make sure no package version number has a `^` character in front of it as it can lead to a version conflict.
:::

[Override the Webpack config](/docs/webpack) by using [`enableSkia()`](/docs/skia/enable-skia).

```ts twoslash title="picus.config.ts"
import {Config} from '@picus/cli/config';
import {enableSkia} from '@picus/skia/enable';

Config.overrideWebpackConfig((currentConfiguration) => {
  return enableSkia(currentConfiguration);
});
```

:::note
Prior to `v3.3.39`, the option was called `Config.Bundling.overrideWebpackConfig()`.
:::

Next, you need to refactor the [entry point](/docs/terminology/entry-point) file to first load the Skia WebAssembly binary before calling registerRoot():

```ts twoslash title="src/index.ts"
// @filename: ./Root.tsx
export const PicusRoot = () => <></>;

// @filename: index.tsx
// ---cut---
import { LoadSkia } from "@shopify/react-native-skia/src/web";
import { registerRoot } from "picus";

(async () => {
  await LoadSkia();
  const { PicusRoot } = await import("./Root");
  registerRoot(PicusRoot);
})();
```

You can now start using the [`<SkiaCanvas>`](/docs/skia/skia-canvas) in your Picus project.

## Rendering

By default Picus rendering are done on the CPU. Some Skia effects rely on advanced GPU features, which may be slow to run on the CPU depending on the kind of effect you are using. If your Skia export is extremely slow, we found that enabling the GPU via the `--gl=angle` option improves things substantially. Please check out the documentation on [GPU rendering](/docs/gpu).

```sh
picus render Main out/video.mp4 --gl=angle
```
