# @picus/rounded-text-box

Create a TikTok-like multiline text box SVG path with rounded corners

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/rounded-text-box.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/rounded-text-box?minimal=true)

## Installation

```bash
npm install @picus/rounded-text-box --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/rounded-text-box) for more information.
