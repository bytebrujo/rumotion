# @picus/enable-scss

Enable SCSS support in Picus

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/enable-scss.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/enable-scss?minimal=true)

## Installation

```bash
npm install @picus/enable-scss --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/enable-scss/overview) for more information.
