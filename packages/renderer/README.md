# @picus/renderer

Render Picus videos using Node.js or Bun

[![NPM Downloads](https://img.shields.io/npm/dm/@picus/renderer.svg?style=flat&color=black&label=Downloads)](https://npmcharts.com/compare/@picus/renderer?minimal=true)

## Installation

```bash
npm install @picus/renderer --save-exact
```

When installing a Picus package, make sure to align the version of all `picus` and `@picus/*` packages to the same version.
Remove the `^` character from the version number to use the exact version.

## Usage

See the [documentation](https://www.picus.dev/docs/renderer) for more information.
